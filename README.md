### Датасет для классификации

- **Название:** Predicting Manufacturing Defects Dataset 
- **Ссылка:** [https://www.kaggle.com/datasets/rabieelkharoua/predicting-manufacturing-defects-dataset](https://www.kaggle.com/datasets/rabieelkharoua/predicting-manufacturing-defects-dataset)  
- **Описание:** В датасете представлены производственные параметры: объем выпуска, стоимость, качество поставщика, задержки поставок, показатели энергоэффективности, число инцидентов безопасности и др.  
Целевая переменная — **DefectStatus**, которая показывает, является ли деталь дефектной.
- **Обоснование выбора:**  
  Выявление дефектных деталей - важная задача из области промышленности. Прогнозирование брака позволяет снизить затраты, уменьшить переработки и улучшить контроль качества. Датасет подходит для моделирования реального процесса контроля качества.

### Датасет для регрессии

- **Название:** Seoul Bike Sharing Dataset (UCI)  
- **Ссылка:** [https://archive.ics.uci.edu/dataset/560/seoul+bike+sharing+demand](https://archive.ics.uci.edu/dataset/560/seoul+bike+sharing+demand)  
- **Описание:** Предсказание спроса на аренду велосипедов на основании часовых данных.  
- **Обоснование выбора:**  
  Датасет включает погодные параметры (температура, влажность, скорость ветра), а также временные признаки (час суток, день недели, сезон) для прогнозирования количества арендованных велосипедов. Актуальность: помогает предсказывать нагрузку на сервисы проката велосипедов, планировать ресурсы и оптимизировать работу городского транспорта.  

### Выбор метрик качества

- **Классификация:**  
  - **Accuracy** — доля правильно классифицированных объектов, базовая и понятная метрика.  
  - **F1-score** — гармоническое среднее precision и recall, важно при несбалансированных классах.  
  - **Confusion Matrix** — для визуального анализа ошибок между классами.

- **Регрессия:**  
  - **Mean Absolute Error (MAE)** — средняя абсолютная ошибка, показывает среднюю величину отклонения предсказаний от фактических значений.  
  - **Mean Squared Error (MSE)** — среднеквадратичная ошибка, штрафует большие ошибки сильнее.  
  - **R² (коэффициент детерминации)** — показывает, какая доля вариации зависимой переменной объясняется моделью.
